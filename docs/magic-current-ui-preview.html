状态控制

Magic 交互细节

Joystick on 时红点为红色并可控制鼠标；按下时红点变大。进入滑动模式后红点变黄。Mic on 时红点变灰，频谱随输入变化。

Mic 图标状态

Magic 页面只显示一个 mic 图标：关闭时灰色，开启时绿色。BtnA 切换时颜色同步变化。

交互流程

操作行为
BtnB 短按Setup 与 Magic 相互切换。
BtnA 短按仅 Magic 生效：Mic off / Joystick on 与 Mic on / Joystick off 互切。
BtnB 长按 3s进入可发现/配对状态，底部状态变为 Pairing。
BtnB 长按 8s清除 bonding 并模拟重启回 Setup。
HFP ChanSetup 上显示实际 HFP audio channel：HFP Chan: ON/OFF。
